# GCD-calculator-c
A gcd calculator written by c 
this project is a simple gcd calculator for two integers. Also known as EBOB in turkish.
## How to run
you need a C compiler for this program
1.**Compile the file** open your terminal and write this commands:

'''bash
gcc gcd.c -o gcd_program
'''
2.**Execute the program**
'''bash
./gcd_program
'''
*the program will prompt you to enter two numbers.*
